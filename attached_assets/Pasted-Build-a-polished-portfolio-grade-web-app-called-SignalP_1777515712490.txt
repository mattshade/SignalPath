Build a polished portfolio-grade web app called SignalPath.

Tagline:
Designing AI-assisted decisions for moments that cannot wait.

The app should be a detailed mobile web journey map for an AI-assisted decision-making workflow. It should show how a product, engineering, or operations leader uses a mobile web experience to respond to a high-stakes situation, understand context, evaluate AI recommendations, make a decision, and follow through.

The goal is to create a portfolio piece that demonstrates:
- Mobile UX strategy
- Journey mapping
- AI interaction design
- Systems thinking
- Trust and control in AI workflows
- High-fidelity product design
- Responsive design and mobile-first craft

Build with React, TypeScript, Tailwind, and local mock data only. No backend or API keys.

Visual direction:
Use a Squarespace-inspired aesthetic without copying logos or exact brand assets.

Default the app to light mode.

The design should feel:
- Editorial
- Premium
- Minimal
- Spacious
- Typographically strong
- Highly crafted
- Calm but sharp
- Design-led rather than dashboard-heavy

Style cues:
- Warm white and soft neutral background

- Generous whitespace
- Strong layout rhythm
- Minimal borders
- Refined cards
- Elegant data visualization
- Subtle motion

Mobile optimization:
This app must be excellent on mobile web.

Design mobile-first, then scale to tablet and desktop.

Mobile requirements:

- No horizontal overflow
- Mobile-first journey timeline
- Embedded mobile UI preview
- Smooth transitions between stages

Core concept:
SignalPath presents a journey map for a scenario called Critical Launch Response.

Scenario:
A product leader is away from their desk when a high-risk issue appears during a major feature launch. They open a mobile web experience that uses AI to summarize the issue, show relevant signals, recommend options, explain tradeoffs, and help them make a confident decision.

Primary user:
A product or engineering leader responsible for launch quality and customer impact.

Journey stages:
1. Alert
2. Triage
3. Context
4. AI Recommendation
5. Trust Check
6. Decision
7. Follow-through
8. Learning Loop
